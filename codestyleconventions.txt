Function Names:
following spatstat as best I can:
- no camel case
- acronyms/abbreviations are lower case
- things which have a capital in the name - eg. K-function will have capital in the function name
- periods in function names are ok

Xi and xi are the RACS and the observed pattern respectively.
Use a _ to separate an object and a facet/property of the object. 
E.g. Xi_H for the H function of Xi.
Use a period to separate a particular data format for an object (e.g. .im)
